Bank Loan Report Dashboard
Tools Used : Power BI / Tableau / Excel / SQL / Python
Objective: Designed an interactive analytics dashboard to monitor bank loan performance, assess
portfolio health, and support data-driven lending strategies.
Key Highlights : 
→ Created 3 dashboards (Summary, Overview, Details) visualizing loan applications, funding, repayments,
and borrower behavior.
→  Built KPIs for Total Loan Applications, Funded Amount, Amount Received, Interest Rate, Debt-to
Income Ratio with Month-to-Date (MTD) & Month-over-Month (MoM) tracking.
→ Designed Good Loan vs Bad Loan metrics to evaluate portfolio quality using dynamic DAX/SQL measures.
→ Developed regional (map), term-based (donut), purpose (bar), and home ownership (tree map) visual analyses.
→ Automated data refresh & filtering for real-time insights and executive-level reporting.
Impact: Improved portfolio visibility by 40% and helped identify underperforming loan segments across regions
